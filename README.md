# A C++ Implementation of a Multi-Start Variable Neighborhood Search (MS-VNS) Algorithm

## Overview

This repository contains a **C++ implementation of a Multi-Start Variable Neighborhood Search (MS-VNS)** algorithm developed for solving a ** MVRDC problem**.



## Why use the MVRDC Instance File

The algorithm needs an **input instance**.

An instance file typically contains the information required by the algorithm, such as:
- node identifiers,
- coordinates,
- demands,
- service times,
- time windows,
- vehicle-related parameters,
- and other problem-specific data that are available in each instance (READ.ME. txt)

Then copy/past the content of each instance in INPUT.txt and run the algorithm.
